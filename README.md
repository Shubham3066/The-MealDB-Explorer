# The-MealDB-Explorer
TheMealDB Explorer is a web application for exploring meals and recipes using TheMealDB API. It follows a clean client-server architecture with a custom backend for API handling and caching, and a responsive, modern frontend for an intuitive user experience.
Step 1: Install Prerequisites
Make sure the following are installed on your system:
Node.js (version 16 or above), npm, VS Code, and the Live Server extension for VS Code.

Step 2: Clone the Repository
Clone the project repository from GitHub and open it in VS Code.

Step 3: Backend Setup
Open a new terminal in VS Code.
Navigate to the Backend folder.
Run npm install to install all required dependencies.
Start the backend server using npm start.
The backend will run on http://localhost:5000

Step 4: Frontend Setup
Open the Frontend folder in VS Code.
Right-click on index.html and select “Open with Live Server”.

Step 5: Use the Application
Click on any category to view meals.
Search meals by name using the search bar.
Click “I’m Feeling Hungry” to get a random meal with full recipe details.

Step 7: Stop the Application
To stop the backend, press Ctrl + C in the terminal.
To stop the frontend, click “Stop Live Server” in VS Code.
